Tensor * MaxPool2D_Forward ( Tensor * input)

Aplica una operación Max Pooling 2D sobre un tensor.

Para cada región de pooling, se selecciona el valor máximo, reduciendo las dimensiones espaciales del tensor de entrada.

Devuelve
Tensor resultante después de la operación de Max Pooling.
